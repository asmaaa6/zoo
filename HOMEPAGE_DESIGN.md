## 📱 RESPONSIVE DESIGN

### Mobile (< 480px)
```
Hero: Full width, stacked
Text: Réduit (2rem pour titre)
Cards: Largeur 100%
Navigation: Hamburger menu
```

### Tablet (480px - 768px)
```
Hero: Still stacked
Cards: 2 colonnes
Text: Moyen
Navigation: Hamburger menu
```

### Desktop (> 768px)
```
Hero: Côte à côte avec image
Cards: 3 colonnes
Text: Normal
Navigation: Menu complet
```

### Large Desktop (> 1600px)
```
Container: Max-width respecté
Spacing: Plus généreux
Fonts: Légèrement plus gros
```

## ✨ ANIMATIONS IMPLÉMENTÉES

### Entrée d'Éléments
```
@keyframes fadeInUp
From:    opacity: 0; translateY(30px)
To:      opacity: 1; translateY(0)
Duration: 0.8s ease-out
```

### Flottaison Cards
```
@keyframes float
0%, 100%: translateY(0px)
50%:      translateY(-20px)
Duration: 3s ease-in-out
Repeat:   Infini
```

### Hover Effects
```
Cards Feature: translateY(-10px), shadow augmentée
Tech Cards:   translateY(-8px), shadow augmentée
Module Cards: translateX(10px), shadow augmentée
Buttons:      translateY(-3px), shadow plus prononcée
```

### Scroll Animations
```
Intersection Observer:
- Fade in au défilement
- Trigger lors visible
- Unobserve après animation
```
